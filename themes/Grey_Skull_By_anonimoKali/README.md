Breaking the darkness with the third official theme for **Bruce Firmware**. **Grey Skull** is a high-visibility, light-mode theme designed for maximum clarity in bright environments without sacrificing that gritty, underground aesthetic.

With its bone-white background and deep black "distressed" typography, this theme is perfect for those who want a clean yet aggressive look.

### ✨ Highlights
* **High-Visibility Light Mode:** Optimized for outdoor use and bright environments.
* **Dual Skull Design:** Features detailed skull icons flanking the main menu text for a unique "memento mori" style.
* **Sharp Contrast:** The black-on-white palette ensures every menu item is readable at a glance.

### 📱 Supported Versions (Included in ZIP):
- **105px:** M5StickC Plus / Plus 2
- **140px:** T-Embed CC1101 / Plus
- **180px:** CYD (Cheap Yellow Display) & T-Deck
- **192px:** T-LoRa-Pager

### 📦 What's Inside?
The attached `.zip` file contains:
- Full folder hierarchy for easy drag-and-drop.
- `Grey_Skull.json` configuration files.
- All required monochrome assets and icons.

### 🚀 Installation
1. Download `Grey_Skull_By_anonimoKali.zip` below.
2. Extract the archive to your computer.
3. Copy the folders into the `/themes` directory of your SD card.
4. Reboot your device and select **Grey_Skull** from the Theme settings.
