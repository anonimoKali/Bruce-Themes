Every hacking tool needs its "Matrix" moment. **Total Tools Green** is a professional-grade theme for **Bruce Firmware** that focuses on a technical, terminal-inspired aesthetic. 

Featuring bright neon green accents and a circuit-style border, this theme is designed to make your device look like a dedicated hardware hacking console.

### ✨ Highlights
* **Terminal UI:** High-contrast green on black for that classic "hacker" look.
* **Tech-Borders:** Custom corner frames that enhance the display's technical feel.
* **Function Preview:** The main menu layout is optimized to look like a live command-line tool.
* **OLED Friendly:** Perfect for battery saving on M5Stack and T-Embed devices.

### 📱 Supported Versions (Included in ZIP):
- **105px:** M5StickC Plus / Plus 2
- **140px:** T-Embed CC1101 / Plus
- **180px:** CYD (Cheap Yellow Display) & T-Deck
- **192px:** T-LoRa-Pager

### 📦 What's Inside?
The attached `.zip` file contains:
- Complete directory structure for all device resolutions.
- `Total_Tools_Green.json` configuration files.
- Exclusive terminal-style assets and icons.

### 🚀 Installation
1. Download `Total_Tools_Green_By_anonimoKali.zip` below.
2. Extract the archive on your PC.
3. Move the folders into the `/themes` directory of your SD card.
4. Go to **Settings > Themes** and select **Total_Tools_Green**.
