Take your **Bruce Firmware** experience to the next level with the **Pro Plus** version of my green-terminal aesthetic. This theme introduces a more complex and "rugged" UI, featuring a new central HUD-style tag and enhanced tactical borders.

Designed for clarity and a professional "cyber-tool" feel, it’s the perfect balance between high-tech visuals and field-tested readability.

### ✨ Highlights
* **Advanced HUD:** New background plate for the main menu titles, giving a 3D tactical look.
* **Cyber-Tech Frame:** More detailed corner accents for a full "encapsulated" interface.
* **Optimized Glow:** The neon green palette is tuned to look spectacular on both M5Stack LCDs and T-Embed screens.
* **Pro Layout:** Clean sub-text for quick identification of tools (WiFi Atks, Sniffers, etc.).

### 📱 Supported Versions (Included in ZIP):
- **105px:** M5StackC Plus / Plus 2
- **140px:** T-Embed CC1101 / Plus
- **180px:** CYD (Cheap Yellow Display) & T-Deck
- **192px:** T-LoRa-Pager

### 📦 What's Inside?
The attached `.zip` file contains:
- The full directory structure for instant SD card setup.
- `Pro_Plus_Tools_Green.json` configuration files.
- All high-resolution tactical assets and UI frames.

### 🚀 Installation
1. Download `Pro_Plus_Tools_Green_By_anonimoKali.zip` below.
2. Extract the contents on your PC.
3. Drag and drop the folders into the `/themes` directory on your SD card.
4. Go to **Settings > Themes** on your device and select **Pro_Plus_Tools_Green**.
