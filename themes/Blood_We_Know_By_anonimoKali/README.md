The second official theme for **Bruce Firmware** is here. **Blood We Know** is a raw, aggressive, and high-impact theme designed for those who want their device to look like a piece of experimental survival gear.

Featuring a "Biohazard/Distressed" aesthetic, this theme uses deep blacks and vibrant blood-orange accents for perfect visibility in any environment.

### ✨ Highlights
* **Gritty Aesthetic:** Custom distressed typography for main menu items.
* **Iconic Look:** Features the signature "Bloody Hand" icon for a unique tactical feel.
* **High Contrast:** Perfectly optimized for M5StickC Plus/Plus2 screens and T-Embed displays.

### 📱 Supported Versions (Included in ZIP):
- **105px:** M5StickC Plus / Plus 2 (Optimized for Portrait)
- **140px:** T-Embed CC1101 / Plus (Optimized for Wide display)
- **180px:** CYD (Cheap Yellow Display) & T-Deck
- **192px:** T-LoRa-Pager

### 📦 What's Inside?
The attached `.zip` file contains:
- The complete folder structure for all resolutions.
- `Blood_We_Know.json` configuration files.
- All high-contrast image assets.

### 🚀 Installation
1. Download `Blood_We_Know_By_anonimoKali.zip` below.
2. Extract the files on your PC.
3. Transfer the contents to the `/themes` folder on your SD card.
4. Select **Blood_We_Know** from your device's Theme menu.
