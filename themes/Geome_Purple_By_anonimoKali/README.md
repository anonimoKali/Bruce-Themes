I am proud to present **Geome Purple**, a high-contrast, geometric theme designed for **Bruce Firmware**. 
This theme is built for hackers and enthusiasts who want a clean, futuristic, and "stealth-neon" aesthetic for their portable tools.

### ✨ Highlights
* **Cyber-Vibe:** Neon Purple accents on a deep black background for maximum OLED/LCD efficiency.
* **Geometric Design:** Sharp, clean lines and custom-styled menu icons.
* **Tested on Hardware:** Fully verified on **LilyGO T-Embed CC1101** (see preview images).

### 📱 Supported Versions included in the ZIP:
This release includes optimized versions for the following screen heights:
- **105px:** M5StickC Plus / Plus 2
- **140px:** T-Embed CC1101 / Plus
- **180px:** CYD (Cheap Yellow Display) & T-Deck
- **192px:** T-LoRa-Pager

### 📦 What's Inside?
The attached `.zip` file contains the complete folder structure:
- `/themes/[resolution]/Geome_Purple.json`
- All required image assets for each device version.

### 🚀 Quick Install
1. Download the `Geome_Purple_By_anonimoKali.zip` below.
2. Extract the content to your computer.
3. Copy the files into the `/themes` folder on your SD card.
4. Go to **Settings > Themes** on your Bruce device and select **Geome_Purple**.
