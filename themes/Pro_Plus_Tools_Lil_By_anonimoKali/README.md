Expanding the **Pro Plus Series**, I’m excited to introduce the **Lil (Purple Neon)** edition. This theme combines the advanced "HUD-style" UI with a vibrant lilac/purple palette, designed for hackers who want a modern, high-tech look for their **Bruce Firmware** setup.

This version features the improved central tag and tactical borders, providing a professional and unified experience across all supported devices.

### ✨ Highlights
* **Cyber HUD Design:** Features the advanced central UI plate for a 3D tactical effect.
* **Lilac Neon Palette:** Deep purples and bright lilac accents for a futuristic terminal feel.
* **Pro Layout:** Optimized icons and sub-text for fast tool navigation (WiFi, BLE, RF, etc.).
* **Eye-Catching Contrast:** Specifically tuned for OLED and LCD clarity in dark environments.

### 📱 Supported Versions (Included in ZIP):
- **105px:** M5StackC Plus / Plus 2
- **140px:** T-Embed CC1101 / Plus
- **180px:** CYD (Cheap Yellow Display) & T-Deck
- **192px:** T-LoRa-Pager

### 📦 What's Inside?
The attached `.zip` file contains:
- The full directory hierarchy for quick SD card deployment.
- `Pro_Plus_Tools_Lil.json` configuration files.
- All high-resolution lilac tactical assets and UI frames.

### 🚀 Installation
1. Download `Pro_Plus_Tools_Lil_By_anonimoKali.zip` below.
2. Extract the archive on your computer.
3. Move the folders into the `/themes` directory on your SD card.
4. Select **Pro_Plus_Tools_Lil** from the Bruce Theme menu.
