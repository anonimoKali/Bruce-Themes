Moving into the 2.0 series with a fresh look! **Gradient Grey** introduces subtle shading and gradients to the **Bruce Firmware** UI. This theme moves away from flat colors to embrace a more sophisticated, "brushed metal" aesthetic.

The grey-to-black transitions provide a sense of depth that makes icons and text pop without being overwhelming. It’s professional, sleek, and incredibly easy on the eyes.

### ✨ Highlights
* **Gradient Effects:** Soft grey-scale transitions for a premium, non-flat UI experience.
* **Modern Depth:** New design elements that add a 3D feel to the main menu.
* **Refined Contrast:** Optimized grey tones to ensure readability while maintaining a stealthy look.
* **Premium Hardware Match:** Designed to look stunning on the T-Embed and M5Stick, enhancing the build quality of your device.

### 📱 Supported Versions (Included in ZIP):
- **105px:** M5StickC Plus / Plus 2
- **140px:** T-Embed CC1101 / Plus
- **180px:** CYD (Cheap Yellow Display) & T-Deck
- **192px:** T-LoRa-Pager

### 📦 What's Inside?
The attached `.zip` file contains:
- Organized folders for all pixel heights.
- `Gradient_Grey.json` configuration files.
- Exclusive gradient-styled assets and UI frames.

### 🚀 Installation
1. Download `Gradient_Grey_By_anonimoKali.zip` below.
2. Extract the archive to your computer.
3. Copy the folders into the `/themes` directory on your SD card.
4. Go to **Settings > Themes** and select **Gradient_Grey**.
