**Border Simply** is the synthesis of structure and clarity. Designed for those who love a minimalist approach but need a clear visual boundary for their UI elements, this theme uses a crisp, thin border to frame the interface, creating a "clean-room" diagnostic look.

It’s the most refined version of my minimalist series yet—perfectly balanced, highly readable, and aesthetically sophisticated.

### ✨ Highlights
* **Framed UI:** A delicate, pixel-perfect border that frames the menu, giving it a structured, professional look.
* **Balanced Minimalism:** Strips away all visual noise, leaving only the essential information clearly marked.
* **Industrial Precision:** The font and alignment are tuned to look like a high-end lab diagnostic tool.
* **High Contrast:** Pure white-on-black design for zero eye strain, even during long sessions.

### 📱 Supported Versions (Included in ZIP):
- **105px:** M5StickC Plus / Plus 2
- **140px:** T-Embed CC1101 / Plus
- **180px:** CYD (Cheap Yellow Display) & T-Deck
- **192px:** T-LoRa-Pager

### 📦 What's Inside?
The attached `.zip` file contains:
- Fully optimized folder structure for rapid deployment.
- `Border_Simply.json` configuration files for every resolution.
- Exclusive border-aligned assets and icons.

### 🚀 Installation
1. Download `Border_Simply_By_anonimoKali.zip` below.
2. Extract the archive to your computer.
3. Move the folders into the `/themes` directory on your SD card.
4. Go to **Settings > Themes** and select **Border_Simply**.
