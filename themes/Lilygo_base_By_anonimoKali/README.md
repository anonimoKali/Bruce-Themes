Introducing **Lilygo Base**, a theme designed for clarity, precision, and a clean professional look. Moving away from neon accents, this release focuses on a high-contrast monochrome palette that perfectly complements the clear casing of modern tactical hardware.

With its sleek white typography and dotted circular UI elements, it provides a "factory-fresh" feel while maintaining the powerful capabilities of **Bruce Firmware**.

### ✨ Highlights
* **Clean Monochrome UI:** Pure white on deep black for the highest possible contrast ratio.
* **Dotted Gauge Design:** Features a unique dotted circular progress/selection indicator on the main screen.
* **Minimalist Iconography:** Simplified white icons for WiFi, Bluetooth, and Config menus.
* **Hardware Focused:** Designed specifically to look spectacular through transparent enclosures and under bright white LEDs.

### 📱 Supported Versions (Included in ZIP):
- **105px:** M5StickC Plus / Plus 2
- **140px:** T-Embed CC1101 / Plus (Primary device)
- **180px:** CYD (Cheap Yellow Display) & T-Deck
- **192px:** T-LoRa-Pager

### 📦 What's Inside?
The attached `.zip` file contains:
- The full directory structure optimized for Bruce Firmware.
- `Lilygo_base.json` configuration files for all supported pixel heights.
- Exclusive monochrome UI assets and circular gauge elements.

### 🚀 Installation
1. Download `Lilygo_base_By_anonimoKali.zip` below.
2. Extract the content to your PC.
3. Move the folders into the `/themes` directory on your SD card.
4. Open your device, navigate to **Settings > Themes**, and select **Lilygo_base**.
