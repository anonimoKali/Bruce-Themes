Introducing **White Line**, the tenth official theme for **Bruce Firmware**. This design is an evolution of the minimalist aesthetic, introducing thin structural lines that frame the UI for a more organized and technical feel.

It's the perfect choice for users who demand extreme legibility combined with a sleek, modern "linear" design that makes every menu transition feel smooth and professional.

### ✨ Highlights
* **Linear UI Design:** Subtle horizontal lines provide a clear structure to the main menu elements.
* **Ultra-Sharp Contrast:** Brilliant white assets on a pitch-black background for maximum OLED/LCD efficiency.
* **Industrial Aesthetic:** Clean, centered typography that mimics high-end lab equipment displays.
* **Focus Mode:** Zero distractions, focusing entirely on tool selection and status icons.

### 📱 Supported Versions (Included in ZIP):
- **105px:** M5StickC Plus / Plus 2
- **140px:** T-Embed CC1101 / Plus
- **180px:** CYD (Cheap Yellow Display) & T-Deck
- **192px:** T-LoRa-Pager

### 📦 What's Inside?
The attached `.zip` file contains:
- Full directory hierarchy for easy SD card setup.
- `White_Line.json` configuration files for all resolutions.
- Exclusive linear-style white icons and UI assets.

### 🚀 Installation
1. Download `White_Line_By_anonimoKali.zip` below.
2. Extract the archive on your PC.
3. Drag and drop the folders into the `/themes` directory on your SD card.
4. Select **White_Line** from the Bruce Theme settings.
